Best Work

My Hello Kitty Bakery Project is an example of my best work. This project was challenging but it was one of the best outcomes. I created a game to move using the arrows on the bottom left of a key board which moves the Hello Kitty Baker. I am very proud of this specific project because I got to show off my knowledge of how to move not only one but five sprites. I got to use my creative aspect for the sprite’s pictures, the icons, and how my whole game would work. By having an interactive game made this project stand out the most because on my other projects it was just input and I would receive an output while this project requires the user to be a part of it constantly or they would lose the game.

The most difficult part of this work was to get the sprite to continuously move down the screen then go back to the top once it has gone off of screen, but if it touches the Baker then it would go back to the top but have a different x position each time. Also creating all of my functions to go to the next screen because I had to use an && statement so it would correctly transition from screen to screen. Displaying the ‘Lives’ and ‘Pastries’ was also a difficult part because I had many variables and sprites which got me lost and for a good time I was unable to display the updaters. The easiest part of this work was designing all of my screens with the texts and images. I did a template before I did any of my coding which was very helpful and made everything more organized. In this project I used variables, if statements, functions, &&, ==, >=, and many sprites.
